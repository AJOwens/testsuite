TAXONOMY SPECIALIZATION XML SCHEMA README
=========================================

The DITA Open Toolkit 1.1.1 has a bug in the XML Schema definition.

If you are using version 1.1.1, please backup the map.xsd and mapMod.xsd
files in the base schema directory and copy map.xsd and mapMod.xsd 
from this directory (demo/thesaurus/xsd) to the base schema directory.
